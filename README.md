# «Кот Бродского» - markup

## Requirements

 * Node.js 9.11.1
 * Yarn 1.9.4

## Install

```
yarn
```

## Build

**Deploy script:**

```bash
srv.sh
```

**Production build:**

```bash
yarn production
```

- minified assets

**Debug build:**

```
yarn debug
```

- not minified assets
- requires development environment: `.env-dev` file

> Note: you need `.env-dev` file for `yarn dev-build`

## Develop

`dev.sh`

`yarn watch` / `yarn serve`
